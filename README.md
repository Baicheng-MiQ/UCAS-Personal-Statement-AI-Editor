# UCAS-Personal-Statement-AI-Editor

This is a UCAS Personal Statement Editor web app powered by AI.
The goal of this project is to help applicants to UK universities write their statements more efficiently. 

There will be four main functionalities in this web app:
- **Research tool**: help users know what universities are looking for
<img width="1055" alt="research tool screenshot" src="https://user-images.githubusercontent.com/85795754/184552242-414c892a-a959-4600-a27e-b117c59e4f7c.png">

- **Brainstorm tool**: allow users to sort out their mind about what to include in the statement clearly\
*Still working on this, no screenshots available*

- **Construct tool**: get started writing paragraph by paragraph, sentence by sentence. With AI to provide personalised suggestions to each paragraph

<img width="1055" alt="brainstorm tool screenshot" src="https://user-images.githubusercontent.com/85795754/184552337-6b315856-88f9-41e8-aa69-cdd1629285ac.png">

<img width="1055" alt="image" src="https://user-images.githubusercontent.com/85795754/184552531-79edfc17-f515-4d10-8977-11956a57d8d5.png">

## Some techs used in this project
- Powered by GPT-3 and BERT model.
- This repo is currently only for front-end development.
  - Vue 3
  - TailwindCSS
  - Vite
  - TipTap (Editor)
- Firebase (BaaS) + FastAPI
